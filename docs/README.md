## üì± Project Overview

**InstaQueue** is a smart campus queue management system that eliminates physical queues through digital appointments and real-time tracking.

**Stack:**
- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express + TypeScript
- Database: MariaDB
- Cache: Redis
- Real-time: Socket.io
